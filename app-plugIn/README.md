# Frontend — React + Vite

## Instalación

```bash
npm install
npm run dev
```

## Instrucciones de desarrollo

- **Variables y funciones:** `camelCase` → `getUserData`, `handleSubmit`
- **Componentes React:** `PascalCase` → `UserCard`, `LoginForm`
- **Clases CSS:** `kebab-case` → `user-card`, `login-form`

## Tecnologías clave

| Librería | Uso |
|---|---|
| `axios` | Peticiones HTTP al backend |
| `react-router` | Navegación entre vistas |
| `zustand` | Manejo de estado global |
| `yup` | Validación de formularios |
| `tailwind` | Estilos utilitarios |
